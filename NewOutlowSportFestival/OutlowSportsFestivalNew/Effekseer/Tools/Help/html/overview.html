概要

動作環境、必要ランタイム等

編集ツール

  • ShaderModel2.0対応のビデオカードを搭載したWindowsXP以降のPC
  • Microsoft .NET Framework 4(Full)
  • 最新のDirectX エンド ユーザー ランタイム

ランタイム(公式)

  • DirectX9(ShaderModel2.0以上)
  • OpenGL 4.1以上
  • OpenGL ES2.0

インストール、アンインストール

編集ツールは「Bin/」に存在します。レジストリ等の操作は一切しておりません。直接起動するか任意のディレクトリにコピーしてください。
アンインストールする場合はディレクトリごと削除してください。

ランタイムは「Runtime/」に存在します。インクルードが必要なファイルは「Runtime/include/」ライブラリは各自コンパイルをお願いします。

ディレクトリ構成

Bin/ 編集ツール
Help/ ヘルプ
Help/Sample/ ヘルプにて使用しているサンプル
Runtime/ 組み込み用ランタイム
RuntimeSample/ 組み込み用ランタイム組み込みサンプル
Sample1/ エフェクトサンプル

ライセンス

本ソフトウェア及びランタイムのライセンスはMITライセンスとなっております。 また、DirectX Tool Kitを使用しております。

テクスチャ及びエフェクトのデータはEffekseerと共に使用する場合のみ、ご自由にお使いください。

免責事項

本ソフトウェア及びランタイムを使用した際のトラブル等には一切責任を負いかねます。
