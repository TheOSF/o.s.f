Seleneとの連携

概要

Seleneとの連携について説明します。なお、Seleneとこのツールは特に提携しているというわけではないため、バージョンや状況によっては失敗する可能性があります。

サンプル：なし

初めに

Seleneの最新版をご利用するようお願いします。

グラフィックスマネージャーの生成

Effekseerは現在、DirectX9のみしか対応していないため、グラフィックスにはDirectX9を使用するよう指定します。 


Selene::Engine::Graphics::eGraphicsAPI apiTable[] = {
	Selene::Engine::Graphics::GRAPHICS_API_DIRECTX_09,
	Selene::Engine::Graphics::GRAPHICS_API_UNKNOWN
};

if ( !core->CreateGraphicsManager(apiTable) )
{
	return NULL;
}

描画

GetDevicePointerという関数を使用し、デバイスを取得します。 


::EffekseerRenderer::Renderer* renderer 
 = ::EffekseerRenderer::Renderer::Create( (LPDIRECT3DDEVICE9)core->GetManager()->GetDevicePointer(), 最大スプライト数 );

SpriteRenderer::CacheDrawの後等に描画するようにします。Seleneは描画のタイミングがシビアなため、注意しましょう。
